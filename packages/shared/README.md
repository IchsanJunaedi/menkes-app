# SehatKu — Shared Package

Shared Zod schemas, TypeScript types, and constants used across `@sehatku/api`, `@sehatku/web`, and `@sehatku/mobile`.

## Contents

- `src/schemas/` — Zod validation schemas (DTOs)
- `src/types/` — TypeScript type definitions
- `src/constants/` — Shared constants (enums, codes, etc.)

This package is a dependency of all other packages in the monorepo.
